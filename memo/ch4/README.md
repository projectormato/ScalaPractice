## コレクション変換のコツ
* map  
    もとの要素と変換先の要素が1:1対応する時

* flatMap  
    もとの要素ひとつに変換先の要素が0からn個に対応する時

* filter/filterNot  
    もとの要素のうち、条件に合致する(しない)ものを取り除きたい時
    
* take/takewhile/dropRight  
    もとのコレクションの前方からいくつか要素を取り出したい時
    
* drop/dropWhile/takeRight  
    もとのコレクションの後方からいくつか要素を取り出したい時
    
* ++  
    2つのコレクションをつなぎあわせたい時
